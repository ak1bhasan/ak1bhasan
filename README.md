## Projects

### [Movie Recommender System](https://github.com/ak1bhasan/Movie-Recommender-System)
`Python` `scikit-learn` `Pandas` `NumPy`

Hybrid recommendation engine using **content-based filtering** (TF-IDF on genres and descriptions) and **collaborative filtering** (cosine similarity on user rating matrices). The main challenge was handling cases where both approaches returned conflicting recommendations.

---

### [ShopNow — E-Commerce Platform](https://github.com/ak1bhasan/ShopNow)
`Python (Flask)` `MySQL` `HTML/CSS` `JavaScript`

Full-stack e-commerce app with user authentication, product catalog, cart, checkout, and an admin dashboard. Built without a frontend framework and uses raw SQL instead of an ORM — both deliberate choices to work closer to the underlying mechanics.

---

### [Algorithm Visualizer](https://github.com/ak1bhasan/Algorithm-Visualizer)
`Java` `Swing` `AWT`

Desktop application that steps through sorting and searching algorithms visually. Written in Java using Swing/AWT to handle rendering and frame updates directly.

---

### [0/1 Knapsack Visualizer](https://github.com/ak1bhasan/0-1_Knapsack_Visualizer)
`C++` `Dynamic Programming`

Prints the full DP table for the 0/1 Knapsack problem with item-by-item decision tracing. Useful for understanding why a correct DP value still leads to wrong item reconstruction.

---

### [Library Management System](https://github.com/ak1bhasan/FirstProjectMU)
`C` `File I/O`

C program for managing books, authors, and borrow records using file-based storage. Required working directly with record serialization and retrieval without a database layer.
